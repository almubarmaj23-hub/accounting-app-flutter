# 📱 تطبيق المحاسبة المالية

تطبيق أندرويد متكامل لتعلم المحاسبة المالية مبني بـ Flutter

## ✨ المميزات
- 📚 **10 فصول** دراسية محاسبية متكاملة
- 🎬 **17 فيديو** تعليمي من يوتيوب
- 📝 **20+ سؤال** في اختبار تفاعلي
- 🧮 **حاسبة محاسبية** (إهلاك، نسب، تعادل، مخزون)
- 📖 **30+ مصطلح** في القاموس
- 📊 **لوحة تقدم** شخصية مع إنجازات
- 🌙 وضع ليلي كامل
- 🌐 دعم اللغة العربية (RTL)

## 📥 تحميل التطبيق
اذهب إلى [Actions](../../actions) → أحدث workflow ✅ → قسم **Artifacts** → حمّل `accounting-app-apk`

## 🛠️ متطلبات التطوير
- Flutter 3.19+
- Dart 3.2+
- Android Studio / VS Code

## 🚀 تشغيل محلياً
```bash
git clone https://github.com/almubarmaj23-hub/accounting-app-flutter.git
cd accounting-app-flutter
flutter pub get
flutter run
```

## 🏗️ بناء APK يدوياً
```bash
flutter build apk --release --split-per-abi
```
